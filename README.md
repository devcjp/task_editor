# Task manager
Простое desKtop приложение на javaFX.
## Возможности
 - Добавление задачи
 - Добавление заметок
 - Редактирование задач
 - Редактирование заметок
 - Поиск
 - Сортировка
 - Completed state
 - Красивый темный UI
## Стэк
 - Java
 - JavaFx
 - CSS
