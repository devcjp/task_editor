# Task manager
Простое desctop приложение на javaFX.
## Возможности
 - Добавление задачи
 - Добавление заметок
 - Редактирование задач
 - Редактирование заметок
 - Поиск
 - Сортировка
 - Completed state
 - красивый ui
## Стэк
 - Java
 - JavaFx
 - CSS
