# Task manager
Простое desctop приложение на javaFX.
## Возможности
 - Добавление задачи
 - Добавление заметок
 - Редактирование задач
 - Редактирование заметок
 - Поиск
 - Сортировка
 - Completed state
 - Красивый темный ui
## Стэк
 - Java
 - JavaFx
 - CSS
